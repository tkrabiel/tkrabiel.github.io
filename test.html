Sonar Acceptance Tests (SAT/HAT)

Why we perform calibrations: To verify system integration, sensor offsets, and ensure data integrity meets specifications.

The Patch Test Workflow

  • 1. Lever Arms: Physical measurement of the INS/GNSS phase center to the sonar acoustic center.
  • 2. Latency & Pitch: Run reciprocal lines over a distinct slope at varying speeds.
  • 3. Roll: Run reciprocal lines over a flat bottom to detect rotational misalignment.
  • 4. Yaw: Run parallel offset lines over a distinct feature.

Standardized testing tools for Extinction tests and target detection.
